
1.	What is C++? Introduction and Features
C++ is a programming language used to create software, games, and applications.
It was developed by Bjarne Stroustrup.
      It is an improved version of C language.
      Features of C++ 
•	simple and fast 
•	supports Object-Oriented Programming (OOP) 
•	reusable code (no need to write again) 
•	secure (data can be hidden) 
•	works on many systems
2.	What is Encapsulation in C++? Explain
Encapsulation means keeping data safe inside a class and using functions to access it.
       Data is hidden using private
3.	What is Inheritance in C++?
Inheritance means one class uses properties of another class.
       It helps to reuse code.
4.	What is Polymorphism in C++?
Polymorphism means one name, many forms.
A function can behave differently based on context.
 
ATM Management System (C++)
1. Introduction
This project is a simple ATM Management System developed using C++ for beginners. It simulates basic ATM operations like checking balance, depositing money, and withdrawing money.
2. Objectives of the Project
To understand basic C++ programming concepts
To learn class and object usage
To implement simple banking operations
To build a console-based application

3. Methodology / System Design
The system is designed using Object-Oriented Programming.
A class ATM is created
It contains:
PIN verification
             Balance management
             Menu system
Flow:
1.	User enters PIN
2.	If correct → menu is displayed
3.	User selects operation:
             Check Balance
             Deposit
             Withdraw
             Exit




4. Implementation / Source Code
•	Language: C++
•	Concepts Used:
o	Classes & Objects
o	Functions
o	Loops (do-while)
o	Conditional Statements
The program starts by verifying the PIN and then provides a menu-driven interface for ATM operations.
5.Conclusion
This project helps beginners understand the basics of C++ and how real-world systems like ATMs work. It can be further improved by adding features like multiple users, file storage, and security enhancements.
